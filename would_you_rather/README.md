# Would you rather

This app lets you play well known game called Would you rather. You can ask any question, you can answer any question and you can see stats of each question. It is also possible to open players' leaderboards.

## TL;DR

To get started developing right away:

* install all project dependencies with `npm install`
* start the development server with `npm start`
